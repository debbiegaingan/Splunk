Discord Webhook Setup Guide

This guide will help you set up a Discord webhook for your login page. Follow these steps to get your webhook URL and configure your login page.

1 Create a Discord Webhook

First, you need to create a webhook in your Discord server:

  1. Open Discord and go to the server where you want to receive login notifications
  2. Right-click on a text channel and select "Edit Channel"
  3. Go to "Integrations" > "Webhooks" > "New Webhook"
  4. Give your webhook a name (e.g., "Login Notifier") and optionally set an avatar
  5. Click "Copy Webhook URL" to copy the webhook URL
  6. Click "Save" to create the webhook
Note: You must have the "Manage Webhooks" permission in the Discord server to create webhooks.

3 Test Your Login Page

After configuring your webhook URL:

  1. Open the index.html file in your web browser
  2. Enter test credentials in the login form
  3. Click the "Login" button
  4. Check your Discord channel to see if the login information was received

4 Deploy Your Login Page

To make your login page accessible to others, you need to host it online. Here are some options:

Option 1: Use the included Node.js server

  1. Make sure Node.js is installed on your computer
  2. Double-click the start-server.bat file
  3. Your login page will be available at http://localhost:3000

Option 2: Use a web hosting service

  1. Sign up for a web hosting service (many offer free plans)
  2. Upload all the files to your web hosting service
  3. Your login page will be available at the URL provided by your hosting service
